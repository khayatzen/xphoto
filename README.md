xphoto
======

XPhoto - Extention for Yii Framework to upload photo and capture from webcam or camera using HTML5

Requirements:
============
- Twitter Bootstrap
- Yii 1.1 above


How to use it:
================
$this->widget('ext.xphoto.XPhoto',array(
 	'model'=>$model,
 	'attribute'=>'user_photo',
));
